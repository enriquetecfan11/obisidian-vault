# Mara System — Overview

## Objetivo
Tener un sistema multiagente dirigido por **Mara** para ejecutar trabajo especializado por rol, con trazabilidad en repositorio y operación práctica diaria.

## Agentes
- **Mara** 👑: orquestadora y decisora final.
- **Atlas** 📅: operaciones personales (calendario, notas/tareas).
- **Arvis** 🧠: creatividad y generación/optimización de contenido.
- **Scout** 🔎: investigación, exploración y síntesis de información.

## Principio operativo
1. Quique define objetivo.
2. Mara decide qué agente actúa.
3. Agente especializado ejecuta.
4. Mara valida, consolida y entrega.

## Repositorios
- **mara-os**: panel/mission-control y capa operativa.
- **mara-knowledge**: memoria, conocimiento, prompts, skills y datasets.
