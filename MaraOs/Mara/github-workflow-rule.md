# Regla operativa — GitHub

## Instrucción de Quique
Siempre que Quique pida que se cambie algo, después de aplicar los cambios se debe **subir a GitHub** por defecto.

## Política práctica
1. Aplicar cambio.
2. Verificar (build/test si aplica).
3. Commit con mensaje claro.
4. Push al repositorio correspondiente.

## Excepciones
- Si Quique indica explícitamente no subir todavía.
- Si hay bloqueo de permisos/credenciales (avisar y resolver).
